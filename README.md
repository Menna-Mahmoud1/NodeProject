# 🌱 Garden API

A RESTful API built with **Node.js**, **Express**, and **MongoDB** to manage a digital garden — including plants, gardeners, and harvests.

---

## 🚀 Features

- CRUD operations for:
  - 🌿 Plants
  - 🧑‍🌾 Gardeners
  - 🌾 Harvests
- JSON-based REST API
- MongoDB Atlas integration
- Modular structure using Routes and Controllers

---

## 📦 Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- dotenv

---
